GUIDownloader

GUIDownloader es una aplicación de escritorio de código abierto para descargar vídeos de YouTube.

Esta aplicación escrita en Python se encarga de descargar cualquier vídeo de YouTube en formato MP4 o MP3. El usuario puede elegir qué formato descargar, así como su calidad. Para descargar los vídeos, la librería yt_dlp, actúa en el backend de la aplicación validando el enlace y recogiendo los metadatos necesarios, posteriormente, al incrustar yt_dlp a Python podremos descargar dicho vídeo en el formato que deseemos con una simple función para más información consulta sobre yt_dlp. A la hora de convertir los formatos de vídeo se usa una aplicación de terceros llamada FFMPEG siendo este uno de los conversores de vídeo más utilizado en la actualidad.

En conclusión, diré que desarrollé esta aplicación porque me interesaba la idea de poder descargar vídeos de YouTube de manera rápida empleando Python. Con este proyecto he aprendido sobre los diferentes formatos de vídeo y audio, así como sus parámetros más importantes.

Tecnologías


  • Python 3.9.1
  • PySide6
  • yt_dlp
  • FFMPEG

Notepy

Notepy es una aplicación de escritorio de código abierto para escribir textos.

Esta aplicación escrita en Python te permite escribir documentos de texto de la misma manera que el bloc de notas tradicional de Windows. Las funciones de esta aplicación son las mismas que el de un bloc de notas, puedes escribir, modificar y guardar archivos .txt donde y cuando tú quieras. Para hacer la interfaz gráfica, utilicé una librería de Python llamada PySide6. Tengo planes de ampliar las funcionalidades de esta aplicación en un futuro y ofrecer temas de colores y más herramientas de edición de texto.

Para concluir diré que desarrollé esta aplicación para aprender cómo funciona y cómo se hace una aplicación de bloc de notas empleando Python como lenguaje de programación.

Tecnologías


  • Python 3.9.1
  • PySide6

Ecommerce

Moonlight es una página web desarrollada para aprender como implementar un Ecommerce.

Esta página web tenía el propósito de poder vender todos tus productos, así como poder editar sus precios y ofertas. En esta web podías registrarte e iniciar sesión como usuario y posteriormente acceder a tu perfil y cambiar sus parámetros. En la página principal de la tienda tenías acceso a todos los productos disponibles, los cuales podrías filtrar por categoría, había un carrito de la compra donde aparecían todos los productos que te interesan comprar y una página "Checkout" donde proceder con el pago.

También utilicé Django para desarrollar esta web, ya que quería aprender a implementar el rol de Ecommerce a una aplicación web hecha con Python-Django a la vez que utilicé Bootstrap para implementar un diseño rápido y moderno. Por desgracia, este proyecto no recibirá más actualizaciones.

  • Django
  • HTML5
  • CSS3
  • Bootstrap
  • JavaScript

CSVToAccess

CSVToAccess es un script para operar con bases de datos .accdb.

Este simple script escrito en Python se encarga de escribir los datos de un archivo CSV a una tabla de Microsoft Access. Este script utiliza librerías como sqlalchemy y pandas para leer los archivos y escribir los datos correctamente al archivo .accdb.

Es importante saber que las columnas entre el archivo CSV y la tabla de la base de datos han de coincidir para que pueda funcionar. En caso de que te dé un error no esperado, es posible que tengas que instalar el motor de base de datos de MS Access.

Tecnologías


  • Python 3.9.1
